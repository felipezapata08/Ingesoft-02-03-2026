## HU-01 – Registro de Paciente

**Como** visitante  
**Quiero** registrarme como paciente  
**Para** poder agendar citas médicas  

### Criterios de Aceptación

1. El sistema no debe permitir registrar un documento de identidad ya existente.
2. El sistema debe validar que el correo tenga formato correcto.
3. El sistema debe mostrar mensaje de registro exitoso.
